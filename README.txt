Para compilar y verificar la práctica se ha facilitado un Makefile. Para compilar hay que hacer los siguiente:
    - make 

make compilará y creará directamente un enlace simbólico en el directorio del Makefile para poder usar el binario.
No se recomienda ejecutar los procesos de forma individual, sino utilizando make, para ello hacer lo siguiente:
    - make test
